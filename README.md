# Parkinson Wearable Digital Biomarkers

A reproducible research-grade baseline pipeline for detecting Parkinson's freezing-of-gait events from wearable accelerometer time-series.

## Project Goal

Build a subject-aware machine learning pipeline for wearable accelerometer data, including preprocessing, windowing, signal-derived features, leakage-safe validation, class-imbalance analysis, calibration metrics, and cautious non-clinical digital-biomarker interpretation.
